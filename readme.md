# WebGL Demos

Mostly simple things done with Three.js.

![WebGL demos](screenshot.jpg)

## Usage

Install the dependencies:

    npm install

...and Bower (wait for it):

    bower install

Build it with `intercessor` in `./build`:

    npm run build

Start it:

    npm run start

See it at [localhost:3000](http://localhost:3000).
